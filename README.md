# LAB4
Program tworzy punkty reprezentujące płaszczyzny Bezziera. Wszystko odbywa się 
w jednej pętli. 

-Wczytuje liczbe płaszczyzn z pliku
-Wczytuje pkt reprezentujące jedną płaszczyzne
-Oblicza położenie punktów na płaszczyźnie
-Zapis do pliku

Można w kodzie podmienić pliki z punktami na inne(łyzka, kubek).
Poza tym punkty w pełni prawidłowe, sprawdzone w Paraview.
